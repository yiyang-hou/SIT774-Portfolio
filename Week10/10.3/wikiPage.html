Information

This project foucses on exploring the extended Bootstrap components and features to further refine the presentation of the website to make it look as professional as possible and offer the users with a positive user experience and easy to use interactions. Several features are explored and implemented.

Navbar
navbar, navbar-expand are used to enble responsive collapsing for navigation bar. It is implemented as such: a button with navbar-toggler feature targets an element with collapse class by its ID. The outcome of this feature is that if the size of the window is smaller than a certain point, the nav bar will collapse and turn into a drop-down menu button that contains all the navigation elements. The navigation options will be contained within the drop-down menu is because each of the options has a nav-item class on them which classifies them as a part of the navigation element.
Social media icons
When social media icons are required, class social and fa can be useful. With external style sheet injected from "https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css", specific social media icons can be retrieved (eg. fa-facebook will give you Facebook icon). Further styling can also be done after retrieving the icons with CSS, on this page, the size, colour scheme and mouse hover effect is added to the icons.
Background image for jumbotron
A background image can be put on jumbotron by using CSS style rules .jumbotron{ background: url(agriculture.jpg) no-repeat center center fixed; background-size: cover;} These rules load the image, set it to be "no-repeat" and covers up the entire jumbotron area. The image is also fixed in the center, it doesn't move as the page gets scrolled by the user.
Breadcrumb
On each subpages other than the home page, breadcrumb can be added to offer a convinent functionality. Breacrumb is a feature indicates the current page's location within a website. It indicates a navigational heirachy of pages. Breadcrumb needs to be true to the structure of the website to avoid causing confusions for the users. Breadcrumb is implemented by have breadcrumb class on a ordered list element, each list item contains a link to a specific page and has a breadcrumb-item class on them. The item up the list has higher heirachy. The item that represents the current page doesn't need to link to any page.
Badges
badges are small labels used to count or label elements. This feature can be implemented by have a span tag next to the element you want to label and put a badge class on the span tag. This will enable the badge to apear next to the desired element.
Pagination
Pagination offers a list of pages that allows the users to navigate through, especially useful for long list of contents. Pagitation can be implemented by putting pagitation class on an unordered list , each list item haspage-item class on them, they can be either a previous button, a next button or any specific page number. each of them refers to another page, and the previous button counts down on the index of the page list, while the next button counts up.
